# Contributing to Conduit

First of all, thank you for considering contributing to Conduit! It is people like you that make Conduit a great tool for AI efficiency.

## How Can I Contribute?

### Reporting Bugs
*   Check the [Issues](https://github.com/your-username/conduit/issues) page to see if it hasn't been reported yet.
*   If you can't find an open issue addressing the problem, [open a new one](https://github.com/your-username/conduit/issues/new). Be as specific as possible!

### Suggesting Enhancements
*   [Open a new issue](https://github.com/your-username/conduit/issues/new) and label it as "enhancement".

### Pull Requests
1.  Fork the repo and create your branch from `main`.
2.  Install dependencies: `npm install`.
3.  Set up your local environment: `cp .env.example .env.local` and fill in your Supabase keys.
4.  Make sure your code follows the existing style and type signatures.
5.  If you've added code that should be tested, add tests.
6.  Ensure the test suite passes.
7.  Submit the PR!

## Local Development

```bash
# Clone the repository
git clone https://github.com/your-username/conduit.git

# Install dependencies
npm install

# Run the dev server
npm run dev
```

Happy coding!
